# Vuelet Vuetify Data Table Example

Small Vue 2 + Vuetify v2 project for exploring `v-data-table`.

Features:
- Sortable columns with built‑in sort indicators.
- Per‑column text filters in the first table row.
- Pagination (10 items per page by default).

Scripts:
- `npm run dev` — start Vite dev server.
- `npm run build` — production build to `dist/`.
- `npm run preview` — preview the production build.
